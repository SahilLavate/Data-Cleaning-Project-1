# Data Cleaning & Preparation Project

## Objective
Clean a raw dataset by handling missing values, duplicate records, and formatting inconsistencies.

## Tasks Performed
- Identified missing values
- Replaced blank CouponCode values with "No Coupon"
- Checked for duplicate records
- Verified OrderID uniqueness
- Verified date formats
- Verified numeric and text fields

## Results
- Missing values handled successfully
- No duplicate rows found
- Zero duplicate Order IDs
- Dataset prepared for analysis

## Tools Used
- Microsoft Excel
